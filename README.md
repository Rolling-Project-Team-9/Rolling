개발 기간: 2023. 11.8 ~ 11.22 (2주)

배포 URL: https://rolling-team9.netlify.app/

### 💡프로젝트 소개
- Rolling은 누구나 쉽게 롤링 페이퍼를 만들고, 작성할 수 있는 웹 플랫폼입니다.
- 롤링 페이퍼 생성 버튼을 눌러 롤링 페이퍼를 직접 만들 수 있습니다.
- 롤링 페이퍼에 메세지를 직접 작성하거나 삭제할 수 있습니다.
